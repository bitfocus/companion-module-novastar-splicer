# companion-module-novastar-splicer

Companion module for Novastar H Series video wall splicers (H2, H5, H9, H15, H20) over UDP port 6000, using the Novastar H Series Control Protocol.

See [HELP.md](./companion/HELP.md) for the full action / feedback / preset / variable list, and [LICENSE](./LICENSE).

## Compatibility

- **Companion**: 4.2 and later (built against `@companion-module/base` 1.5; also loads on Companion 4.3+).
- **Hardware**: tested against H5 (other H Series models share the same protocol surface and should work).
- **Protocol**: Novastar H Series Control Protocol V1.0.19.
